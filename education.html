Служба занятости

СПб ГАУ «Центр занятости населения Санкт-Петербурга»

СПб ГАУ релизует специальные мероприятия по содействию трудоустройству инвалидов:

  • трудоустройство инвалидов на квотируемые рабочие места;
  • специализированные ярмарки вакансий для инвалидов;
  • трудоустройство инвалидов на созданные за счет бюджета Санкт-Петербурга рабочие места;
  • создание рабочих мест для трудоустройства инвалидов.

  • посредническая помощь в трудоустройстве инвалидов;
  • call-центр для консультаций инвалидов по вопросам поиска работы;
  • обучение востребованным профессиям;
  • приобретение спеиального оборудования для инвалидов.
